# ModBundle

A Minecraft mod manager for Android, built for **PojavLauncher** and other mobile Minecraft launchers. Search, download, and manage mods, resource packs, and shader packs directly on your phone.

---

## Features

### Browse & Search
- Search mods, resource packs, and shader packs from **Modrinth** and **CurseForge**
- Filter by Minecraft version and mod loader
- Supports **Fabric**, **Forge**, **NeoForge**, and **Quilt**
- Toggle between release versions and snapshots

### Download & Install
- One-tap download directly into your instance's mods folder
- Automatic dependency detection — choose which required or optional dependencies to install alongside a mod
- Supports both modern Android storage (SAF) and legacy file-based storage

### Instances
- Manage multiple Minecraft instances
- Set a custom name, icon, Minecraft version, and loader per instance
- Quickly switch the active instance from the Instances tab

### Installed Mods
- View all mods, resource packs, and shaders installed in the active instance
- Check for updates across all installed mods at once
- Update individual mods or all outdated mods in one tap
